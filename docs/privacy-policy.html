MyNotes - Privacy Policy

Ultimo aggiornamento: 6 febbraio 2026

Informazioni generali

MyNotes e' un'applicazione desktop open source per la gestione di note personali. Il codice sorgente e' disponibile su GitHub.

Dati raccolti

MyNotes non raccoglie, trasmette o condivide alcun dato personale. Tutte le note, allegati e impostazioni sono salvati esclusivamente in locale sul dispositivo dell'utente nella cartella data/ accanto all'applicazione.

Google Drive

MyNotes offre una funzione opzionale di backup su Google Drive. Quando l'utente sceglie di attivare questa funzione:

  • L'app richiede l'accesso tramite OAuth 2.0 con lo scope drive.file, che consente di creare e gestire solo i file creati dall'app stessa.
  • L'app non puo' leggere, modificare o eliminare altri file presenti nel Google Drive dell'utente.
  • I file caricati sono copie di backup del database locale (.db), salvati in una cartella dedicata ("MyNotes Backup").
  • Il token di autorizzazione e' salvato esclusivamente in locale sul dispositivo dell'utente.
  • L'utente puo' revocare l'accesso in qualsiasi momento dalle impostazioni dell'app o dalla pagina Autorizzazioni account Google.

Condivisione dati con terze parti

MyNotes non condivide alcun dato con terze parti. Non esistono server remoti, analytics, tracking o telemetria di alcun tipo.

Sicurezza

Le note possono essere crittografate dall'utente con password tramite crittografia AES con derivazione chiave PBKDF2. Le password non vengono mai salvate.
